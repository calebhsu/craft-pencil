# craft-pencil

Parameterized pencil model.

### Usage
```html
<craft>
    <craft name="pencil" module="craft-pencil"/>
    <pencil></pencil>
</craft>
```

### Parameters
- length: adjusts length of pencil

### Examples
```html
<craft>
    <craft name="pencil" module="craft-pencil"/>
    <row spacing="1">
        <pencil length="20"></pencil>
        <pencil></pencil>
    </row>
</craft>
```
